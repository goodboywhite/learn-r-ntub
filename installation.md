R 語言主程式
========================================================

從 CRAN 下載 R 語言主程式：

<https://cran.r-project.org/>

整合開發環境 RStudio
========================================================

從 RStudio 下載 RStudio：

<https://www.rstudio.com/products/rstudio/download3/>

第一次執行 RSudio
========================================================

- 來源（Source）：位於左上角，編寫程式的區塊
- 命令列（Console）：位於左下角，執行程式的區塊
- 環境與歷史：位於右上角
- 檔案、圖形、套件、查詢與預覽器：位於右下角

第一個函數：`q()`
========================================================

- 在命令列（Console）輸入 `q()` 可以離開 RStudio

其他函數
========================================================

```
ls() # 目前環境中有哪些變數
rm() # 移除環境中的變數
getwd() # 目前的檔案路徑在哪裡
setwd() # 設定檔案路徑
Sys.Date() # 回傳系統日期
Sys.time() # 回傳系統日期與時間
```
